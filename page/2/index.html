2333…..螺一天是40~60mg

螺内酯,学名安体舒通，剂量是4-6粒，服用一段时间后感觉效果还可以，也试了一下最高的剂量14粒，肾开始非常疼差点把我吓坏了，脸和毛发上的油脂也消失了，后来经过推论和网上资料证明，螺内脂的抗雄是副作用，需要大量滴喝水，并且很强肾脏，效果也很慢，急不来。螺内脂，好处是便宜，好买到。副作用……很多……抗雄就是副作用之一，伤肾，去厕所频繁，容易得低血压和高血钾，所以吃螺内脂的时候应该尽量避免大量食用含钾类食物（豆制品，海带之类）。

···
$ npm install hexo-renderer-sass –save
$ npm install hexo-generator-json-content –save
$ npm install hexo-generator-feed –save
